üß© Introduction

Responsive design ensures that a webpage looks and functions well on all screen sizes ‚Äî from phones to desktops. Modern CSS gives us two powerful layout systems to build responsive pages:

  • Flexbox: Ideal for 1D layouts (row or column)
  • Grid: Ideal for 2D layouts (row and column)

This guide will explain how they work and how to use them to create clean, flexible, and responsive web layouts.

üì¶ The Box Model

Every HTML element is a rectangular box made of:

  1. Content ‚Äî text or images inside
  2. Padding ‚Äî space between the content and the border
  3. Border ‚Äî line around the padding
  4. Margin ‚Äî space between the border and other elements

Example:

box-sizing: content-box; /* Default */

box-sizing: border-box; /* Preferred for layouts */

Using border-box makes width/height include padding and borders, which simplifies layout calculations.

üîß Flexbox Basics

Flexbox is designed for 1D layouts ‚Äî either horizontal or vertical.

Activating Flexbox:

.container {

display: flex;

}

Key Properties (on the flex container):

  • flex-direction: row (default), column, etc.

  • justify-content: /* controls alignment along main axis */ (e.g., center, space-between)

  • align-items: /* controls alignment along cross axis */ (e.g., center, stretch)

  • gap: /* space between items */

Key Properties (on flex items):

  • flex-grow: lets items grow

  • flex-shrink: lets items shrink

  • flex-basis: sets initial size before growing/shrinking

  • flex: shorthand for all 3 above

.item {

flex: 1;

}

üß™ Flexbox Examples


  1. Navbar

    HTML:

    <nav class="flex">

    <a>Home</a>

    <a>About</a>

    <a>Contact</a>

    </nav>

    CSS:

    nav {

    display: flex;

    justify-content: space-between;

    }



  2. Centered Card

    .container {

    display: flex;

    justify-content: center;

    align-items: center;

    height: 100vh;

    }



  3. Equal Columns

    .col {

    flex: 1;

    }

üìê Grid Basics

Grid is made for 2D layouts ‚Äî rows and columns.


Activating Grid:

.container {

display: grid;

}


Core Properties

  • grid-template-columns: defines column layout (e.g., repeat(3, 1fr))

  • grid-template-rows: row sizes

  • gap: space between rows/columns

  • place-items: shorthand for align-items + justify-items

.container {

display: grid;

grid-template-columns: repeat(3, 1fr);

gap: 1rem;

}



Grid Item Properties:

  • grid-column: 1 / 3; ‚Üí spans from column 1 to 3

  • grid-row: 2; ‚Üí places item in row 2
üéØ Grid Examples

  1. Simple 3x2 Grid

    .container {

    display: grid;

    grid-template-columns: repeat(3, 1fr)

    grid-template-rows: auto auto;

    }

  2. Image Gallery

    .gallery {

    display: grid;

    grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

    gap: 1rem;

    }

  3. Layout with Header, Sidebar, Main

    .layout {

    display: grid;

    grid-template-areas:

    "header header"

    "sidebar main";

    grid-template-columns: 200px 1fr;

    }

üì± Responsive Design

Responsive design adapts layout for different screen sizes.

Use Media Queries:

@media (max-width: 768px) {

.container {

flex-direction: column;

}

}

Tips:

  • Use relative units (% , em, fr) instead of fixed px

  • Use minmax() with auto-fit or auto-fill for flexible grids

  • Stack elements on small screens using flex-direction or grid-template-areas

Feature Flexbox Grid
Axis One (row/column) Two (row & col)
Best For Navbars, Cards Page Layouts

Use Flexbox for distributing items.

Use Grid when the layout structure is known ahead of time.
